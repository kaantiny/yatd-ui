Keyboard Shortcuts
j/k or ↑/↓ - Navigate
Enter - View task details
z - Collapse/expand selected tree node (or toggle open/closed in flat mode)
Esc - Blur search (when focused)
\ - Close detail panel
/ - Focus search
x - Clear search
o - Show open only
c - Show closed only
a - Show all
t - Toggle tree view
d - Toggle DAG view
y - Copy selected task id
r - Refresh
⌘/Ctrl + / - Toggle project sidebar
? - Toggle this help
